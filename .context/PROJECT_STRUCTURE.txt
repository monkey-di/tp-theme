license.txt               Theme license

CONFIGURATION:
==============

src/input.css
  @import "tailwindcss"
  @font-face (Akrobat, Ura Bum Bum SP)
  @theme { colors, fonts }
  @layer base { typography }

functions.php
  wp_enqueue_style('tebe-poveryat-tailwind', output.css)
  wp_enqueue_script('tp-main', main.js)
  wp_enqueue_script('tp-sliders', sliders.js)
  wp_enqueue_style('swiper-css', swiper.css)
  wp_enqueue_script('swiper-js', swiper.js)

theme.json
  settings.color.palette
  settings.typography.fontFamilies
  settings.layout
  styles.color, styles.typography

BUILD COMMANDS:
===============

Development:  npm run dev       (watch + recompile)
Production:   npm run build     (minified output)
Copy Assets:  npm run copy-assets (Swiper to vendor)

============

Tablet:  768px+   (md:)
Desktop: 1024px+  (lg:)

SWIPER SLIDERS:
===============
1. Hero        (.hero__slider)
2. Friends     (.friends__slider)
3. Media       (.media__slider)
4. Materials   (.materials__slider)
5. Histories   (.histories__slider)
6. Team        (.team__slider)

Each has: autoplay, navigation arrows, progress bar, breakpoints

MOBILE MENU:
============
Trigger:  .js-open-mobile-menu (burger button in header)
Overlay:  .mobile-menu-overlay (fixed overlay from parts/)
Animation: translate-x-full → translate-x-0 (Tailwind classes)
Close:    ESC key or .mobile-menu-overlay__close-btn
Body:     Scroll locked when menu open

COMPONENT SYSTEM:
=================

button.php
  Args: text, url, style, size, state, class, submit
  Styles: primary, secondary, white, outline-primary, outline-white, form-submit
  Sizes: sm, md, lg
  States: default, disabled
  Renders: <a> if url, <button> otherwise

link-more.php
  Learn More style link with arrow icon
  Variations: normal, hero

input-with-button.php
  Email input with embedded button
  Used in footer subscription form

slider-navigation.php
  Prev/Next arrow buttons
  Customizable colors

slider-progress.php
  Swiper progress bar
  Configurable track/bar colors
